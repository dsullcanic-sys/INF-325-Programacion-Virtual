BLOQUES BÁSICOS ANDROID JAVA

1. Seleccionar archivos con PICKER (Android 10+)

Intent intent = new Intent(Intent.ACTION_OPEN_DOCUMENT);
intent.setType("*/*"); // o text/csv, image/*, etc.
intent.addCategory(Intent.CATEGORY_OPENABLE);
startActivityForResult(intent, PICK_FILE_CODE);

Recibir resultado:

@Override
protected void onActivityResult(int requestCode, int resultCode, @Nullable Intent data) {
    super.onActivityResult(requestCode, resultCode, data);
    if (requestCode == PICK_FILE_CODE && resultCode == Activity.RESULT_OK && data != null) {
        Uri uri = data.getData();
        // Ejemplo: leer el archivo
        try (InputStream input = getContentResolver().openInputStream(uri)) {
            BufferedReader reader = new BufferedReader(new InputStreamReader(input));
            String line;
            while ((line = reader.readLine()) != null) {
                Log.d("ARCHIVO", line);
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

21. Estructura recomendada para DBHelper y modelos

public class DBHelper extends SQLiteOpenHelper {
    // ... constructor y métodos ...

    // Modelo interno
    public static class Cliente {
        public int carnet;
        public String nombres;
        public int codProf;
        public int codDepto;
        public Cliente(int carnet, String nombres, int codProf, int codDepto) {
            this.carnet = carnet;
            this.nombres = nombres;
            this.codProf = codProf;
            this.codDepto = codDepto;
        }
    }
    // ... otros modelos ...
}

25. Recomendaciones finales

- Mantén tus queries y modelos en el DBHelper.
- Cada Activity debe instanciar su propio DBHelper.
- Usa funciones para mostrar mensajes y limpiar campos.
- Valida siempre los datos antes de consultar o insertar.
- Guarda este manual y amplíalo con tus propios bloques.
